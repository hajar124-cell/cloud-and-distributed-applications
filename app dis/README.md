## Concepts abordés

- **Architecture distribuée** : modèles client-serveur, peer-to-peer, architecture n-tiers
- **Communication** : RPC, sockets, messages asynchrones, protocoles distribués
- **Coordination** : synchronisation des processus, exclusion mutuelle, horloges logiques
- **Tolérance aux pannes** : redondance, réplication, mécanismes de consensus
- **Scalabilité** : partitionnement, load balancing, microservices
